# Email (not bundled in this SPA)

This **Vite static build** does not include a Node server or transactional email SDK.

- **Contact visitors** can use `mailto:` links or social channels from **`siteConfig`**.
- To send email from forms, add a **backend** (serverless function, API route, or third-party form endpoint) and point your form `fetch()` there.

For a full-stack reference with **Resend** on Express, see separate templates; this repository stays frontend-only.
